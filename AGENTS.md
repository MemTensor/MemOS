# AGENTS.md

This file is provided for AI coding harnesses that look for `AGENTS.md` by
convention (Codex, Cursor, OpenHands, Aider, etc.).

The harness engineering specification for this repository — project
overview, common commands, branch / commit / PR conventions, code style,
testing rules, and harness-specific guardrails — lives in
**[CLAUDE.md](./CLAUDE.md)** at the repository root.

**Please read `CLAUDE.md` end-to-end before making changes.** Everything in
it applies to all agentic tools regardless of vendor; the filename is
historical.

Human contributors should start from
**[CONTRIBUTING.md](./CONTRIBUTING.md)** instead — it covers fork / clone /
PR flow in more depth.
